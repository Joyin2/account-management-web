Logout Functionality Test

To test the logout button:

  1. Go to http://localhost:3000
  2. Click "Login" and sign in with valid credentials
  3. After successful login, you should see:
    • "Dashboard" tab in the navigation
    • User menu with your name/email
    • "Logout" option in the dropdown menu
  4. Click on the user menu and then "Logout"
  5. Verify that:
    • You are redirected to the home page
    • Navigation shows "Login" and "Sign Up" buttons again
    • No user-specific content is visible

Expected Behavior:

  • ✅ Logout button should be visible in user menu when logged in
  • ✅ Clicking logout should sign out the user
  • ✅ User should be redirected to home page
  • ✅ Navigation should revert to showing Login/Signup buttons
  • ✅ User menu should disappear
